# GCSV - CSV Viewer

A simple GUI application for viewing CSV files built with Go and Fyne.

## Project Information

- **Date of Creation**: June 22, 2025
- **Total Hours Worked**: 3 hours
- **Language**: GoLang

## Usage

```bash
go run src/*.go <path-to-csv-file>
```

## Dependencies

- Fyne v2.6.1 (GUI framework)
